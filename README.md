# OCR

A Full-stack OCR implementation.

## Features

- Upload pdf
- Pipeline to get md files
- OpenAPI docs exposed by the backend

## Tech stack

- **Frontend:** React + TypeScript + Vite + Tailwind
- **Backend:** Node.js + TypeScript + Trpc
- **Database:** PostgreSQL (dribble ORM)
- **Pubsub/queue:** Redis + Rabbitmq
- **Monorepo:** pnpm workspaces
